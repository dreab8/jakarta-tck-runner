Jakarta Persistence Technology Compatibility Kit, Version 2.2
Release Notes, September, 2019

Kit Contents

The Jakarta Persistence, Version 2.2 Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta Persistence TCK tests signature, API, and End-to-End tests:

    • A signature test that checks that all of the public APIs are supported in the Jakarta Persistence Version 2.2 implementation that is being tested
    • API tests to verify the functionality of all the public APIs under
      • javax.persistence
      • javax.persistence.criteria
      • javax.persistence.metamodel
      • javax.persistence.spi
    • End-To-End tests that demonstrate compliance with Java Persistence API 2.2

Platform Notes

The Jakarta Persistence TCK tests have been built and tested with JavaTM Platform, Standard Edition 8 (Java SE 8).

The Jakarta Persistence TCK tests have been run on the following platforms:

  • Windows 10
  • Oracle Linux 7.1
